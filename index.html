Welcome to the INF 122 Web Programming Course Website. This website has been developed to provide students with a comprehensive overview of the first three modules covered in the course. The contents are organized to help learners understand the fundamental concepts of web development and gain practical knowledge of website creation.

The website is divided into three major modules:

  • Module 1: Web Development Introduction
  • Module 2: Hyper Text Markup Language (HTML)
  • Module 3: Cascading Style Sheets (CSS)
 I hope this website serves as a useful learning resource and helps students develop the skills required for modern web development.


Course Overview

INF 122 is an introductory Web Programming course for 100 Level Information Systems students. The course introduces students to the technologies used in developing modern websites.

About This Website

This website was developed by DAVID VICTOR as part of the INF 122 (Web Programming) course assignment for 100 Level Information Systems students. The website provides educational content on the first three modules taught in the course.

Each module contains relevant units, explanations, examples, images, tables, forms, and multimedia resources to enhance learning and understanding of web development concepts. The website also includes navigation links that allow users to move easily between modules and a contact page for visitor feedback and communication.
