Unit 1: Introduction to CSS

CSS CSS (Cascading Style Sheets) is the design and layout language used to style websites. CSS controls its visual appearance, including colors, fonts, spacing, and responsive design across different screens.

This box is styled using CSS.


Unit 2: The Three Main Ways to Write CSS

Inline CSS: Applied directly to an HTML tag using the style attribute. Best for quick, single-element adjustments.

Internal CSS: Written in a block placed in the section of your HTML document. Great for styling a single page

External CSS: Written in a separate file (e.g., styles.css) and linked to your HTML using links tag. This is the industry standard for full websites.



Unit 3: CSS Selectors

  • Element/Tag Selector: Targets all instances of a specific HTML tag (e.g., p, h1, body).


  • Class Selector: Targets elements with a specific class attribute, denoted by a dot (.) (e.g., .nav-link, .highlight). Used when you want to style multiple unique elements the same way.


  • ID Selector: Targets a single, specific HTML element denoted by a hash (#) (e.g., #main-header). IDs must be unique on a page.
