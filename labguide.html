Introduction

This Lab Guide provides practical exercises and instructions for students taking INF 122 (Web Programming). The purpose of the laboratory sessions is to help students gain hands-on experience in designing and developing web pages using HTML and CSS.

Laboratory Objectives

  • Understand the fundamentals of web development.
  • Create and structure web pages using HTML.
  • Apply CSS for webpage styling.
  • Insert images, tables, forms and multimedia.
  • Develop simple and responsive websites.

Laboratory Requirements

Item Purpose
Computer System For writing and testing web pages.
Text Editor VS Code, Notepad++ or any code editor.
Web Browser Google Chrome, Firefox or Edge.
Internet Access For research and deployment.

Lab Activities

Exercise 1: Creating a Simple HTML Page

Create a webpage containing:

  • A heading
  • A paragraph
  • An image
  • A hyperlink

Exercise 2: Creating Lists and Tables

Create ordered and unordered lists and design a table with rows and columns.

Exercise 3: Designing Forms

Create a form that collects Name, Email and Comments from users.

Expected Learning Outcomes

At the end of the laboratory exercises, students should be able to design and develop simple websites, structure content using HTML, apply styles using CSS and publish web projects online.

Lab Safety Rules

  • Handle computer systems carefully.
  • Save work regularly.
  • Avoid unauthorized software installation.
  • Maintain a clean laboratory environment.
  • Follow instructions provided by the lecturer.
